idf-runtime

Data plane для IDF домена. SPA bundle — временный placeholder (M1.1); полный V2Shell доставляется в M1.2 после разблокировки Vite build.

Работающие endpoints:

  • GET /health — liveness
  • GET /admin/health — HMAC-signed
  • POST /admin/reload — signed ontology push
  • GET /admin/audit?since=&entity=
  • GET /admin/rejected
  • POST /api/effects — Bearer JWT + viewer middleware
  • GET /api/voice/:slug/:projection
  • GET /api/document/:slug/:projection
  • GET /api/agent/:slug/{schema,world}
